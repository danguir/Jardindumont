Create Virtual Tours for Your Site That Empowers Your Visitors And Clients

== Description ==

iPanorama 360 is the WordPress plugin out there that lets you create awesome virtual tours for clients from directly inside the WordPress admin in seconds. The plugin supports hotspots for providing information about any part of the scene or for navigation to other rooms/areas. It uses their own tooltip system, you can enrich it with text, images, video and other online media. Use this plugin to create interactive tours, maps and presentations.

Using **[ipanorama id="123"]** shortcode, you can publish a virtual tour on any Page or Post in your WordPress sites.


== Installation ==

1. Upload the entire **ipanorama** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Create a new virtual tour, Use **[ipanorama id="123"]** shortcode to publish your virtual tour into any page or post.
